# 진(秦) 제국: 통일과 몰락의 기록 — 교육용 비주얼노벨 RPG

중학교 세계사/한국사 수업용으로 제작된 15~20분 분량의 비주얼노벨 RPG입니다.
순수 HTML + CSS + JavaScript로만 제작되어 외부 라이브러리 없이 동작하며,
GitHub Pages에 폴더 그대로 올리면 바로 실행됩니다.

## 1. 실행 방법

- **GitHub Pages**: 이 폴더 전체를 저장소에 올리고 Pages 기능을 켜면 `index.html`이 바로 열립니다.
- **로컬 테스트**: 이미지 로딩 문제를 방지하려면 `file://`로 직접 열기보다,
  VSCode의 Live Server 확장 등 간단한 로컬 서버를 이용해 여는 것을 권장합니다.

## 2. 폴더 구조

```
index.html      화면 구조(타이틀/설정/게임/엔딩)
style.css       디자인 전체 (모바일·PC 반응형, Fade 애니메이션)
save.js         localStorage 저장/이어하기
scenario.js     대사·선택지·능력치 변화 데이터 (교사가 가장 많이 수정할 파일)
ending.js       엔딩 판정 로직 + 실제 역사 비교 텍스트
ui.js           타이핑 효과, 능력치 바 애니메이션, 이미지 Fade 전환
script.js       전체 게임 진행을 지휘하는 메인 로직
images/bg/      배경 이미지
images/npc/     인물(NPC) 이미지
images/event/   (선택) 특별 이벤트 삽화용 - 현재 시나리오에서는 미사용, 필요 시 scenario.js에서 npc/bg 자리에 자유롭게 사용 가능
sfx/            효과음 mp3 파일 (없어도 오류 없이 무음 처리됨)
```

## 3. 이미지/삽화 교체하기

`images/bg`, `images/npc` 폴더에는 현재 자리표시용 SVG 그림이 들어 있습니다.
실제 삽화(jpg, png, svg 무엇이든 가능)로 교체하려면:

1. 원하는 이미지를 같은 폴더에 넣습니다.
2. `scenario.js` 안에서 해당 장면의 `bg:` 또는 `npc:` 경로를 새 파일명으로 바꿉니다.
   예) `bg: "images/bg/bg_throne.svg"` → `bg: "images/bg/함양궁.jpg"`
3. 이미지가 없거나 경로가 틀려도 게임이 멈추지 않고 조용히 비워서 표시합니다.

## 4. 시나리오(대사/선택지) 수정하기

`scenario.js` 파일 상단의 주석에 씬(scene) 객체 형식이 설명되어 있습니다.
- 대사만 바꾸고 싶다면 `text:` 값만 수정하면 됩니다.
- 선택지를 추가/삭제하려면 `choices:` 배열의 항목을 늘리거나 줄이면 됩니다.
- 각 선택지의 `effects:` 는 능력치 변화량입니다.
  - 공개 능력치: `wangkwon`(황권), `minsim`(민심), `gunsa`(군사), `jaejeong`(재정)
  - 숨김 능력치: `gwijok`(귀족 불만), `baekseong`(백성 불만), `yusaeng`(유생 반발)

## 5. 엔딩 조건 수정하기

`ending.js`의 `calculateEnding()` 함수 안 숫자 조건만 바꾸면
엔딩이 갈리는 기준(예: 몇 점 이상이면 성군 엔딩인지)을 자유롭게 조정할 수 있습니다.
`ENDINGS` 객체의 텍스트를 바꾸면 실제 역사 비교 설명도 원하는 내용으로 교체할 수 있습니다.

## 6. 학생 활동 아이디어

- 플레이 후 자신이 받은 엔딩과 "실제 역사 비교" 내용을 읽고, 실제 역사적 인물의 정책을
  조사해 발표하기
- 같은 챕터에서 서로 다른 선택을 한 친구와 결과 비교하기
- 능력치 변화 로그(개발자 도구 콘솔에서 `gameState.history` 확인 가능)를 바탕으로
  "나의 선택이 어떤 결과를 낳았는가" 성찰 일지 쓰기
